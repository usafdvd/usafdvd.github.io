The Math class helps to perform mathematical operations on numbers. The .random() is a static method that returns a double, between 0 and 1. This can be used to generate a pseudorandom number.
Math is capitalized because it is a class that comes within the JDK. Therefore you do not have to create an object of type Math, and the .random is a static method within this class. Hopefully I will be able to explain this in more depth soon.
